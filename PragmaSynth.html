Pragmatic Synth

Make a Synthesizer

We make a simple synthesizer VCF - VCO - VCA models.

UI

The graphical UI omitted because the description is huge, use the following in a simple interface.

  • T("keyboard") gets input from the keyboard (line:5)
  • T("ndict.key") converts from a key code to a MIDI note number (line:6)
  • T("midicps") converts from a MIDI note number to a frequency (line:8)
