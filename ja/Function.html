English日本語

T(Function)

KR FunctionWrapper

Description

関数を格納して bang() で実行します。関数の結果を自身値として出力します。

Properties

  • func (Function)

    • 格納している関数

  • args (Array)

    • 関数に渡す引数

Methods

  • bang(...)
    • 関数を実行します。戻り値が数値の場合は自身の出力値を更新します。

Note

  • bang() に引数がある場合, bang() の引数 + .args プロパティが関数に渡される引数となります
